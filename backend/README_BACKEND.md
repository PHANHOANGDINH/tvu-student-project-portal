\# Backend - TVU Student Project Portal



\## Công nghệ sử dụng



\- Node.js

\- Express.js

\- SQL Server

\- JWT Authentication

\- dotenv

\- mssql



\## Chức năng đã hoàn thành



\### Sprint 1 - Authentication

\- Đăng ký tài khoản

\- Đăng nhập

\- Sinh JWT token

\- Phân quyền User/Admin



\### Sprint 2 - Quản lý câu hỏi

\- Thêm câu hỏi

\- Xem danh sách câu hỏi

\- Xem chi tiết câu hỏi

\- Cập nhật câu hỏi

\- Xóa mềm câu hỏi



\### Sprint 3 - Quản lý câu trả lời

\- Thêm câu trả lời

\- Xem câu trả lời theo câu hỏi

\- Cập nhật câu trả lời

\- Xóa mềm câu trả lời



\### Sprint 4 - Vote và Tag

\- Vote câu hỏi

\- Vote câu trả lời

\- Quản lý thẻ/tag

\- Gắn tag cho câu hỏi



\### Sprint 5 - Bình luận và tìm kiếm

\- Bình luận câu hỏi

\- Bình luận câu trả lời

\- Tìm kiếm câu hỏi



\### Sprint 6 - Admin và Profile

\- Dashboard admin

\- Quản lý người dùng

\- Quản lý câu hỏi

\- Quản lý câu trả lời

\- Quản lý bình luận

\- Trang hồ sơ người dùng
